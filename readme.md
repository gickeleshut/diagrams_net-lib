# Library for Custom Shapes/Icons for Draw.io

This is a library for custom shapes to be used in the program (Draw.io)[https://github.com/jgraph/drawio]

## Library Categories

* Fiber ST/LC Patch Panels
* Cat5/5e/6 Patch Panels
* Cable Management
* UPCs
* More to come
